Getting Started

Welcome to the Warehouse SaaS Platform! This guide will help you understand how to use our platform to generate detailed AutoCAD site plans from aerial images.

Basic Workflow

  1. Upload an aerial image of your warehouse site
  2. Set the real-world dimensions of the area shown in the image
  3. Select which features you want to detect (railroads, drainage, utilities, etc.)
  4. Generate your site plan
  5. Download the results in your preferred format (DXF, DWG, PDF)

Uploading Images

For best results, please follow these guidelines when uploading aerial images:

Recommended Image Specifications

  • Format: JPEG, PNG, or GIF
  • Resolution: At least 1200 x 1200 pixels
  • Size: Maximum 10MB
  • Clarity: Clear, well-lit images with good contrast
  • Angle: Top-down aerial view preferred

Pro Tip

Images from Google Earth, drone photography, or satellite imagery work best. Ensure the image is taken during daylight hours with minimal cloud cover for optimal feature detection.

Feature Detection

Our AI-powered system can detect various features in your warehouse site images:

Railroads

Rail lines, spurs, and loading areas

Drainage Systems

Drainage channels, retention ponds, and water management systems

Utilities

Power lines, utility boxes, and service connections

Outdoor Furniture

Benches, tables, and other outdoor fixtures

Fences/Barriers

Security fencing, barriers, and perimeter structures

Trees/Vegetation

Trees, shrubs, and landscaped areas

Select only the features you need for your site plan. Processing time increases with the number of features selected.

Setting Dimensions

Accurate dimensions are crucial for creating properly scaled AutoCAD files:

Required Measurements

  • Width: The horizontal measurement of the area shown in the image
  • Height: The vertical measurement of the area shown in the image
  • Length: The depth measurement for 3D representations
  • Unit: Select meters, feet, or yards based on your preference

Important

If you don't know the exact dimensions, you can estimate them using Google Maps or other mapping tools. Inaccurate dimensions will result in incorrectly scaled AutoCAD files.

DWG Native AutoCAD format with full feature support AutoCAD users
PDF Portable Document Format for viewing and sharing Presentations and documentation

Troubleshooting

Common issues and their solutions:

Image upload fails

Check that your image meets our size and format requirements. Try reducing the file size or converting to a different format.

Features not detected correctly

Try using a clearer image with better lighting and contrast. Some features may be difficult to detect if they're obscured or too small in the image.

Download button not working

Ensure your browser allows downloads from our site. Try using a different browser if the issue persists.

Processing takes too long

Processing time depends on image size, complexity, and the number of features selected. Try selecting fewer features or using a smaller image.

Frequently Asked Questions

How accurate is the feature detection?

Our AI system typically achieves 85-95% accuracy depending on image quality and feature clarity. Results may require minor manual adjustments for perfect precision.

Can I edit the generated files?

Yes, all downloaded files can be edited in AutoCAD or other compatible CAD software.

Is there a limit to how many projects I can create?

Standard accounts can create up to 10 projects per month. Premium accounts have unlimited project creation.

How long are my projects stored?

Projects are stored for 30 days on standard accounts and indefinitely on premium accounts.

Can I use satellite imagery?

Yes, satellite imagery works well as long as it has sufficient resolution and clarity.
